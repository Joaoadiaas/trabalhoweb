Sobre Mim

Olá! Sou João Pedro Dias, um entusiasta de tecnologia e desenvolvimento web. Estou atualmente cursando Ciência da Computação e buscando oportunidades para aplicar meus conhecimentos em projetos desafiadores e inovadores.

Projeto 1

O Projeto 1 foi uma aplicação web desenvolvida para gerenciamento de tarefas. Foi construída utilizando HTML, CSS e JavaScript, permitindo aos usuários criar, editar e excluir tarefas de forma simples e intuitiva.

Projeto 2
